## ☁️ Microsoft Defender for Cloud

**Category:** Cloud Security Posture Management (CSPM) & Workload Protection

### Purpose
Secures **cloud infrastructure and workloads**, primarily in Azure.

### What it protects
- Virtual Machines
- Databases
- Storage Accounts
- Containers and Kubernetes
- Networking resources

### Core capabilities
- Secure Score and security recommendations
- Misconfiguration detection
- Vulnerability assessment
- Threat detection on cloud workloads

### Scope
- Subscription and resource level
- Managed via Azure Portal

### Integration
- Sends workload-related alerts to **Microsoft Defender XDR**

### Key point
**Focuses on cloud infrastructure security, not users or endpoints.**

### Defender for Cloud Overview Dashboard

![Defender-for-cloud](screenshots/M.S-Defender-for-Cloud.png)

This dashboard provides a high-level view of the security posture of the Azure subscription.
It highlights:
- Secure Score
- Active security recommendations
- Current security alerts

This view allows security teams to quickly assess risk exposure across cloud resources.
